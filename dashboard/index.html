About This Project

This enterprise-grade customer analytics platform demonstrates advanced data engineering capabilities by processing multi-source customer data to deliver actionable business intelligence. Built entirely on free-tier services, it showcases a production-ready serverless architecture that would typically cost $9,500/month commercially.

The data pipeline integrates customer records from S3 data lakes, RDS MySQL transactional databases, and DynamoDB real-time behavior streams. Using Alteryx Designer for ETL orchestration and AWS Lambda for serverless processing, the system performs customer segmentation, churn prediction (75% accuracy), and CLV calculations. The analytics engine processes RFM (Recency, Frequency, Monetary) scoring, behavioral pattern analysis, and risk stratification to identify $42K in at-risk revenue.

Data flows through a four-stage pipeline: ingestion from multiple sources, Alteryx ETL transformation, Lambda-based analytics processing, and dual visualization via Power BI and this web dashboard. API Gateway exposes RESTful endpoints for programmatic access, while CloudWatch monitors costs in real-time with automated alerts at 80% budget threshold.
